Investigation Overview¶

my main goal is to see the effect of covid-19 (quarantine) on the system by comparing after and before data for the same month (2019 - 2020)

Dataset Overview¶

The dataset contain data about ride renting system each row represent a ride and columns give details about each ride (time (year , month and day), duration , location)
